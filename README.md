# zabbix-event-generator
Навайбкожено.

Довольно часто нужно поморгать триггерами.
Желательно чтобы не править условие как дураку.
Можно и скриптом конечно но с интерфейсом как-то наряднее.

Ни в коем случае не выставляйте наружу.

Как поднять:
1) docker build -t zabbix-gen .
2) docker run -d -p 8080:5000 -e ZABBIX_SERVER="адрес сервера или прокси Zabbix" zabbix-gen
3) Импортировать шаблон в Zabbix (Собрано для 6.0)

Возможно буду расширять инструмент по мере необходимости
